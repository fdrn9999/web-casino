# 🎰 베가스 — 가상머니 라이브 카지노

**현금 결제가 일절 없는 가상머니(칩) 전용** 한국어 실시간 라이브 카지노 웹 앱입니다.
충전·환전·머니이전 같은 **현금 관련 기능은 설계상 존재하지 않으며**, 오히려 도박의 위험성을
알리는 교육 목적을 겸합니다 — 파산 시 누적 손실을 직면시키고, 1시간마다 휴식을 권장하며,
마이페이지와 명예의 전당에서 손실을 숨김없이 보여줍니다.

> 본 사이트는 가상머니 전용입니다. 실제 도박은 오락이 아닌 손실이며, 중독은 질병입니다.

## 빠른 시작

### 요구 사항
- Node.js 20+ (개발/검증은 Node 24)

### 개발 실행
```bash
npm install
npm --prefix server install
npm --prefix client install
npm run dev          # 서버 :4000 + 클라이언트 :5173
```
브라우저에서 **http://localhost:5173** 접속.

### 프로덕션 실행
```bash
npm run build        # client/dist 생성
npm start            # http://localhost:4000 (Express가 정적 파일까지 서빙)
```

### 기본 계정
- **관리자**: `admin` / `admin1234` (서버 최초 기동 시 자동 생성, `server/.env`로 변경 가능)
- 일반 유저는 회원가입으로 생성 (가입 시 10,000칩 지급)

> 참고: 서버 포트는 일부 Windows 환경의 예약 포트 범위를 피해 **4000**입니다. 게임 데이터는
> `server/data/casino.db`(SQLite, git 제외)에 저장됩니다. 깨끗한 상태로 시작하려면 이 파일을 지우세요.

## 아키텍처 개요

- **서버 권위(Server-authoritative)**: 잔액·카드·룰렛 결과·타이머·정산을 **전부 서버가 결정**합니다.
  클라이언트는 서버 스냅샷을 렌더링하고 입력만 보냅니다. 승패를 클라이언트가 계산하지 않습니다.
- **잔액 단일 경로**: 모든 칩 증감은 `server/src/services/wallet.js`의 `applyTransaction` 하나만 거칩니다.
  SQLite 트랜잭션으로 원자 처리하고 `transactions` 테이블에 전부 기록 → 이 기록이 통계·마이페이지·명예의 전당의 원천입니다.
- **게임 루프**: 블랙잭·룰렛·바카라는 관리자가 만든 테이블마다 서버 상태 머신(Runner)이 상시 순환합니다
  (`베팅 → 진행/연출 → 정산 → 다음 라운드`). 참가자가 0명이면 휴면, 첫 입장 시 즉시 재개.
- **실시간**: Socket.IO 기본 네임스페이스(잔액·공지·잭팟·테이블 목록·강제 차단)와
  게임별 네임스페이스(`/blackjack`, `/roulette`, `/baccarat`) + 테이블별 room(`table:{id}`).
- **재시작 복원력**: 서버가 라운드 중 종료됐다면, 다음 기동 시 미정산 베팅을 자동 환불합니다(원자적, 이중환불 방지).

## 유저 기능 (상세)

### 1. 회원가입 · 로그인
- 회원가입: 아이디(영문소문자·숫자·밑줄 4~20자) / 비밀번호(8자+) / 닉네임(12자 이내) + **가상머니·도박 위험성 고지 동의(필수)**.
  가입 즉시 **10,000칩** 지급.
- 로그인: JWT 발급(localStorage 보관). 새로고침해도 로그인 유지.
- **차단된 계정**은 로그인 거부 + 접속 중이면 즉시 강제 종료(게임 소켓 포함).

### 2. 이코노미 (칩 획득 수단 — 모두 무료, 현금 없음)
- **가입 축하 칩**: 10,000칩 (설정 가능).
- **출석 보너스**: 하루 1회(KST 자정 기준) 1,000칩.
- **📅 출석부(연속 출석 보상)**: 로비의 출석부 캘린더에서 연속 출석 시 보상이 증가
  (1→7일차: 1,000 / 1,500 / 2,000 / 2,500 / 3,000 / 4,000 / 5,000칩, 7일 이후 유지). 하루 거르면 리셋.
- **파산 구제**: 잔액이 구제 기준(기본 100칩) 미만이면 신청 가능. 신청 시 **누적 손실액·파산 횟수를 보여주는 경고 모달**을 확인해야 3,000칩 지급 + 5분 쿨다운. 파산 횟수는 통계에 집계.

### 3. 로비
- 게임별 **테이블 목록**(관리자가 만든 오픈 테이블) — 테이블 이름·실시간 착석 인원·베팅 한도 표시. 클릭해 입장.
- 열린 테이블이 없으면 "관리자가 테이블을 준비 중입니다" 안내.
- 슬롯머신 입장 카드, 고정 공지 배너, **실시간 잭팟 금액 위젯**(반짝임 연출), 헤더 잔액(카운트업 애니메이션).
- 진입 버튼: 🎁 출석 보너스, 📅 출석부, ⚠️ 파산 구제(잔액 부족 시), 🏆 명예의 전당.

### 4. 게임 4종

#### 🃏 블랙잭 (실시간 멀티플레이어, 7석)
- 원하는 좌석에 앉아 참가. 베팅 타이머 → 딜링 → **좌석 순서대로 개인 턴**(턴 타이머, 만료 시 자동 스탠드) → 딜러 → 정산.
- 액션: 히트 / 스탠드 / 더블 / 스플릿(같은 랭크) / 서렌더(설정 시).
- 딜러 홀카드는 뒤집혀 있다가 딜러 차례에 **플립 애니메이션**으로 공개. 승리 좌석 글로우, 결과 오버레이(블랙잭!/승리/무승부/패배/버스트/서렌더).

#### 🎡 룰렛 (실시간 멀티플레이어)
- 유러피언(0 하나). 번호판에서 **인사이드 베팅**(번호 1·2·3·4·6개 선택) + **아웃사이드 베팅**(레드/블랙·홀/짝·하이/로우·더즌·칼럼).
- 한 라운드에 여러 베팅 가능. 마감 후 **실제 휠이 회전·감속**해 서버 결과 번호에 정확히 정지, 결과 셀 하이라이트, 최근 결과 히스토리.

#### 🀄 바카라 (실시간 멀티플레이어)
- 플레이어(1:1) / 뱅커(0.95:1) / 타이(8:1) / P·B 페어(11:1) 베팅. 표준 드로잉 룰 자동 진행.
- 카드 공개 연출, 승리 사이드 글로우, 결과 배너, 최근 결과 구슬(P=파랑·B=빨강·T=초록).

#### 🎰 슬롯머신 (싱글 + 전역 프로그레시브 잭팟)
- 3릴, **세로 스크롤로 돌아가는 릴**이 왼쪽부터 순차 정지. **오토 스핀** 토글 지원(잔액 부족/잭팟 시 자동 정지).
- 스핀 베팅액의 일정 비율이 **전역 잭팟 풀**에 적립되고 모든 접속자에게 실시간 표시.
- 7-7-7이면 풀 전액 당첨 → **풀스크린 잭팟 연출**(컨페티 + 금액 카운트업 + 화면 흔들림 + 잭팟 사운드) + 전체 유저에게 축하 배너 브로드캐스트.

### 5. 라이브 테이블 공통 UX
- 하단 **HUD 바**: 잔고 / 이번 라운드 베팅액 / 라운드 상태 / 베팅 한도·배당 표기.
- 베팅 카운트다운 타이머(3초 이하 경고음), 칩 선택 UI.
- **💬 테이블 채팅**: 같은 테이블 참가자끼리 실시간 채팅(접기 가능). 서버측 금칙어 마스킹 + 200자 제한 + 도배 방지(2초 간격).
- 재접속 시 테이블 상태 스냅샷 자동 복구.

### 6. 명예의 전당(리더보드) · 마이페이지
- **🏆 명예의 전당**: 💰 부자 랭킹(보유 칩) / 🎉 최대 당첨(단일 최대 지급) / ⚠️ **최대 손실**(교육용 — "도박의 끝은 대부분 손실입니다").
- **👤 마이페이지**: 총 베팅·총 획득·**순손익(손실이면 빨간색 "총 N칩 손실")**·파산 횟수, 최근 30일 일별 순손익 차트, 최근 거래 내역(한글 라벨).

### 7. 사운드 · 반응형
- 버튼·칩·카드 딜·스핀·승패 효과음은 **Web Audio로 합성**(외부 파일 의존 없음), 잭팟만 mp3. 헤더에서 **음소거 토글**(저장됨).
- 모바일/PC **반응형**(가로 스크롤 없음). 모든 연출은 `prefers-reduced-motion`을 존중.

## 관리자 기능 (상세)

관리자(`role=admin`)만 접근하는 `/admin` 화면. **라우트 가드 + 서버 미들웨어 이중 검증**으로 보호됩니다.

### 1. 유저 관리 (`/admin/users`)
- 아이디·닉네임으로 검색, 유저 상세(잔액·최근 거래 50건·파산 횟수).
- **칩 지급**(금액 + 사유 필수) — 지급 즉시 해당 유저 잔액이 **실시간 반영**.
- **몰수**(금액 또는 전액 + 사유 필수) — 잔액 초과분은 잔액까지로 클램프.
- **차단 / 차단 해제**(사유 필수) — 차단 시 로그인 거부 + **접속 중인 모든 소켓(게임 포함) 즉시 강제 종료**. 관리자 계정은 차단 불가.
- 모든 지급·몰수·차단은 `transactions`에 사유와 함께 기록되어 감사·통계에 반영.

### 2. 테이블 관리 (`/admin/tables`)
- 블랙잭·룰렛·바카라 **테이블 생성**(게임 종류 / 이름 / 선택적 베팅 한도).
- 수정(이름·한도), **열기 / 닫기**, 삭제. 닫기·삭제 시 진행 중 라운드의 **미정산 베팅을 전액 환불**하고 참가자를 로비로 내보냄.
- 테이블 목록은 실시간 착석 인원 표시. 서버 기동 시 열린 테이블의 게임 루프가 자동 시작.

### 3. 공지 관리 (`/admin/notices`)
- 공지 작성 / 수정 / 삭제(관리자 전용), 상단 고정(pinned) 지원.
- 등록 즉시 접속 중인 모든 유저에게 **실시간 배너 푸시**. 일반 유저는 조회만.

### 4. 게임 규칙 설정 (`/admin/settings`)
- 그룹별 규칙을 폼으로 조정(값 유효성 검증 포함), 저장 시 "다음 라운드부터 적용" 안내.
  - **이코노미**: 가입 지급·일일 보너스·파산 구제액·구제 기준·쿨다운.
  - **슬롯**: 최소/최대 베팅·베팅 단위·잭팟 적립률·잭팟 시드.
  - **블랙잭**: 슈 덱 수(1~8)·소프트17 히트·서렌더 허용·더블 허용·스플릿 허용·블랙잭 배당(3:2/6:5)·최소/최대 베팅·베팅/턴 시간.
  - **룰렛**: 최소/최대 베팅·베팅 시간·스핀 연출 시간.
  - **바카라**: 최소/최대 베팅·베팅 시간·공개 시간·타이 배당·페어 배당.
- 규칙 변경은 진행 중 라운드에 영향 없이 **다음 라운드부터** 반영.

### 5. 통계 대시보드 (`/admin/stats`)
- 기간 필터(7일 / 30일 / 전체).
- 요약 카드: 총 가입자 · 오늘 활동 유저 · 총 베팅액 · **하우스 손익**.
- 차트(Chart.js): 일별 베팅·지급 추이(라인), 게임별 하우스 손익(바), 파산 구제 추이(바).
- 표: 잭팟 당첨 이력, 베팅 상위 유저(순손익 색상 표기).
- 모든 수치는 `transactions`/`users` 집계(KST 날짜 기준).

## API·소켓 요약

### REST (일부)
| 메서드·경로 | 설명 | 권한 |
|---|---|---|
| `POST /api/auth/signup` · `/login`, `GET /api/me` | 가입·로그인·내 정보 | 공개/로그인 |
| `POST /api/bonus/daily`, `GET/POST /api/relief`, `GET/POST /api/attendance` | 일일 보너스·파산 구제·출석 | 로그인 |
| `GET /api/tables`, `GET /api/notices`, `GET /api/leaderboard`, `GET /api/me/stats` | 테이블 목록·공지·리더보드·내 통계 | 로그인 |
| `GET /api/slots/state`, `POST /api/slots/spin` | 슬롯 상태·스핀 | 로그인 |
| `/api/admin/users`, `/api/admin/tables`, `/api/admin/notices`, `/api/admin/settings`, `/api/admin/stats` | 관리자 기능 | 관리자 |

### Socket.IO
- **기본 네임스페이스**: `balance:update`, `notice:new`, `jackpot:pool`/`jackpot:won`, `tables:update`, `session:banned`
- **게임 네임스페이스** (`/blackjack`·`/roulette`·`/baccarat`, room `table:{id}`):
  클라→서버 `table:join` / `seat:join` / `seat:leave` / `bet:place` / `action` / `chat:send`,
  서버→클라 `table:state`(스냅샷) / `chat:message` / `table:closed`
